# PyScan---Professional-Port-Scanner
Bu proje, Python `socket` ve `concurrent.futures` modülleri kullanılarak geliştirilmiş, çok iş parçacıklı (multi-threaded) bir port tarama aracıdır.
## Özellikler
- **Hızlı Tarama:** ThreadPoolExecutor ile eş zamanlı tarama.
- **Servis Tespiti:** Bilinen portlar için servis isimlerini gösterir.
- **Banner Grabbing:** Açık portlardan servis bilgilerini çekmeye çalışır.
- **Esnek:** Kullanıcı dostu port aralığı ve timeout ayarı.
- **Raporlama:** Sonuçları JSON formatında dışa aktarma.
